# Legend helper for geom_pop icon legends

Legend helper for geom_pop icon legends

## Usage

``` r
scale_legend_icon(size = 10, margin = NULL, ...)
```

## Arguments

- size:

  Numeric. Legend key size in mm (this controls icon size).

- margin:

  ggplot2::margin() for plot margin (defaults to bottom padding).

- ...:

  Additional arguments forwarded to ggplot2::guide_legend()
