# Package index

## Core geoms

The two main plot layers.

- [`geom_pop()`](https://jurjoroa.github.io/ggpop/reference/geom_pop.md)
  : Create a circular representative population chart
- [`geom_icon_point()`](https://jurjoroa.github.io/ggpop/reference/geom_icon_point.md)
  : Create a scatter plot with Font Awesome icons instead of points

## Data preparation

Prepare your data for ggpop.

- [`process_data()`](https://jurjoroa.github.io/ggpop/reference/process_data.md)
  : Process Population Data for Visualization

- [`fetch_df_coordinates()`](https://jurjoroa.github.io/ggpop/reference/fetch_df_coordinates.md)
  :

  Fetches the `df_coordinates_final` Dataset

## Legends & icons

Control legend appearance and discover available icons.

- [`scale_legend_icon()`](https://jurjoroa.github.io/ggpop/reference/scale_legend_icon.md)
  : Legend helper for geom_pop/geom_icon_point legends
- [`marker_legend()`](https://jurjoroa.github.io/ggpop/reference/marker_legend.md)
  : Build a standalone composite legend of icon markers
- [`fa_icons()`](https://jurjoroa.github.io/ggpop/reference/fa_icons.md)
  : Search and list Font Awesome icons
- [`ggpop_markers()`](https://jurjoroa.github.io/ggpop/reference/ggpop_markers.md)
  : List the icon markers ggpop can render by name

## Themes

Built-in ggpop themes.

- [`theme_pop()`](https://jurjoroa.github.io/ggpop/reference/theme_pop.md)
  : Population Plot Theme
- [`theme_pop_dark()`](https://jurjoroa.github.io/ggpop/reference/theme_pop_dark.md)
  : Dark Population Plot Theme
- [`theme_pop_minimal()`](https://jurjoroa.github.io/ggpop/reference/theme_pop_minimal.md)
  : Minimal Population Plot Theme
